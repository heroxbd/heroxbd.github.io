Motivation

When evaluating the quality of ipv6 connection, it is very hard to judge from the official websites of virtual private server (VPS) providers. An explicit list is presented here for your convenience.

Legend

  • Platinum: An ipv6 prefix (<=64) is routed to a VPS.
  • Gold: Within a prefix (<=64), any ipv6 address can be used via ICMP6 neighbor discovery.
  • Silver: Within a prefix (<=64), any ipv6 address can be used via ICMP6 neighbor discovery. But the address pool is shared with others.
  • Bronze: Ipv6 is available but the number of addresses less than a /64 prefix.
  • Garbage: No ipv6 is provided but ipv6 tunnel protocols can be used, e.g. SIT.
  • Shit: Neither ipv6 is provided nor ipv6 tunnel protocols can be used.
